Commands.

help - Shows a help command

helpnsfw - Shows a help command for NSFW commands

ping - Shows the latency of the bot

setdelay [time] - Sets the slowmode delay of the channel it is executed into

autoad [serverid] [channelid] - Allows the bot to advertise a message a selected number of times.

createchannel [channel name] - Creates a channel with the specified name.
